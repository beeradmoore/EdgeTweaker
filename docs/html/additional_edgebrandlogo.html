Configure the organization logo shown on the New Tab Page for iOS and Android

Supported versions

  • Windows: Not supported
  • macOS: Not supported
  • Android: ≥ 143
  • iOS: ≥ 143

Description

This policy controls whether Microsoft Edge for iOS and Android displays your organization's logo in the New Tab Page when users sign in with their work account.

The Banner logo defined in Microsoft Entra branding is used as the organization logo. You need to maintain your organization logo and organization color by following the steps here: Configure your company branding

If you enable this policy, Microsoft Edge shows your organization's logo in the New Tab Page. If you disable or don't configure this policy, Microsoft Edge won't show your organization's logo in the New Tab Page (default behavior).

Supported features

  • Can be mandatory: Yes
  • Can be recommended: No
  • Dynamic Policy Refresh: Yes
  • Per Profile: Yes
  • Applies to a profile that is signed in with a Microsoft account: No

Data type

  • Boolean

Android information and settings

  • Preference Key name: EdgeBrandLogo
  • Example value:

true

iOS information and settings

  • Preference Key name: EdgeBrandLogo
  • Example value:

xml <true/>
