List of disabled Mini Apps (obsolete)

OBSOLETE: This policy is obsolete and doesn't work after Microsoft Edge version 149.

Supported versions

  • Windows: Not supported
  • macOS: Not supported
  • Android: 140-149
  • iOS: Not supported

Description

This policy lets you specify a list of Mini Apps that are disabled in Microsoft Edge. Users can't launch the Mini Apps listed in this policy.

Provide the policy value as a list of strings, with each string corresponding to the Mini App name. The following Mini Apps are supported:

  • 'featured_feature': Featured Feature (Landing page)
  • 'sydney_chat': Copilot
  • 'download_manager': Download Manager
  • 'pdf_reader': PDF Reader
  • 'news_feed': News Feed
  • 'translator': Translator
  • 'converter': Unit Converter
  • 'money': Money
  • 'images': Images
  • 'covid': COVID Info
  • 'games': Games
  • 'deals': Deals Hub
  • 'wallet': Wallet
  • 'weather': Weather
  • 'gaokao': Chinese Gaokao
  • 'weather_etree': Weather Etree
  • 'weather_map': Weather Map
  • 'weather_life': Weather Life

If you don’t configure this policy or leave it empty, all Mini Apps remain available.
