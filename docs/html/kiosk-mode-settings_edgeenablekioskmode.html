Microsoft Edge Browser Policy Documentation EdgeEnableKioskMode | Microsoft Learn

Enable Microsoft Edge kiosk mode on Android

Supported versions

  • Windows: Not supported
  • macOS: Not supported
  • Android: ≥ 143
  • iOS: Not supported

Description

This policy applies to Microsoft Edge for Android when used in kiosk scenarios.

If you enable this policy, Microsoft Edge for Android runs in kiosk mode, restricting the UI to a locked-down experience.

If you disable or don't configure this policy, Microsoft Edge for Android doesn't run in kiosk mode.

Supported features

  • Can be mandatory: Yes
  • Can be recommended: No
  • Dynamic Policy Refresh: No - Requires browser restart
  • Per Profile: Yes
  • Applies to a profile that is signed in with a Microsoft account: No

Data type

  • Boolean

Android information and settings

  • Preference Key name: EdgeEnableKioskMode
  • Example value:

true
