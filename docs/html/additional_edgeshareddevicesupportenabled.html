Microsoft Edge Browser Policy Documentation EdgeSharedDeviceSupportEnabled | Microsoft Learn

Enable Shared Device Support

Supported versions

  • Windows: Not supported
  • macOS: Not supported
  • Android: ≥ 127
  • iOS: ≥ 125

Description

This policy lets you configure support for shared device in Microsoft Edge.

If you enable or don't configure this policy, support for shared device is turned on in Microsoft Edge.

If you disable this policy, support for shared device is turned off, and Microsoft Edge doesn't run in shared device mode even on a shared device.

Note: This policy only takes effect on first run when Microsoft Edge hasn't been initialized, once the device mode is determined, it remains unchanged unless Microsoft Edge is reinstalled.

Supported features

  • Can be mandatory: Yes
  • Can be recommended: No
  • Dynamic Policy Refresh: No - Requires browser restart
  • Per Profile: No
  • Applies to a profile that is signed in with a Microsoft account: Yes

Data type

  • Boolean

Android information and settings

  • Preference Key name: EdgeSharedDeviceSupportEnabled
  • Example value:

true

iOS information and settings

  • Preference Key name: EdgeSharedDeviceSupportEnabled
  • Example value:

xml <true/>
